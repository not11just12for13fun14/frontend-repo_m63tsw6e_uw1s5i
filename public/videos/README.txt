Place your own source videos in this folder and name them as follows:
- hero.mp4 (+ optional poster-hero.jpg)
- p1.mp4 (+ optional poster-p1.jpg)
- p2.mp4 (+ optional poster-p2.jpg)
- p3.mp4 (+ optional poster-p3.jpg)
- p4.mp4 (+ optional poster-p4.jpg)

These files will be used for the autoplaying previews and the hero background.
